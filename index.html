Proyecto foro social


Como dice el titulo, este proyecto consistió en un prototipo de un foro social pero uso exclusivamente para personas que habitan en departamentos, condominios, o para un grupo de personas que habitan un mismo lugar. Surgió a base de querer buscar facilitar la comunicación como en condominios, ya que con la llegada de la pandemia se fueron reduciendo considerablemente las reuniones entre personas de forma presencial, este proyecto buscó facilitar dicha comunicación de forma virtual, que permitiera expresar problemas, descontentos, comunicar, avisar, etc.

Para el backend se utilizó: Java con Springboot.
Para el frontend se utilizó: CSS con Bootstrap.
